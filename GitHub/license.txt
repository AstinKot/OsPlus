ЛИЦЕНЗИЯ ПРОГРАММЫ OS+

1. ОБЩИЕ ПОЛОЖЕНИЯ

1.1 - Настоящая лицензия — юридически обязывающий документ, регулирующий использование ПО OsPlus (далее — Программа).
1.2 - Использование Программы означает полное и безоговорочное согласие с условиями лицензии.
1.3 - Правообладатель Программы — AstinKot.
1.4 - Программа распространяется в следующих формах:
  | Ярлык на рабочем столе (для запуска)
  | Файлы в папке, находящейся в каталоге Program Files
  | Большинство скриптов написаны на "Пакетном файле Windows - .bat"
1.5 - Лицензия предоставляет неисключительное право использования Программы на одном устройстве.
1.6 - Термины «Пользователь», «Вы» означают физическое или юридическое лицо, установившее Программу.
1.7 - Лицензия действует с момента установки Программы.
1.8 - Несоблюдение условий лицензии влечёт её автоматическое прекращение.
1.9 - Правообладатель вправе изменять условия лицензии с уведомлением через репозиторий GitHub.
1.10 - Термин «Устройство» означает один ПК/ноутбук под управлением ОС Windows.

2. ПРАВА И ОБЯЗАННОСТИ СТОРОН

2.1 - Правообладатель обязуется:
  | Предоставлять работоспособную версию Программы
  | Обеспечивать техническую поддержку через GitHub Issues
  | Не вносить изменения, нарушающие заявленный функционал
2.2 - Пользователь обязуется:
  | Использовать Программу только по назначению
  | Не распространять, не модифицировать, не декомпилировать Программу без согласия Правообладателя
  | Не использовать Программу для незаконных целей
  | Не скачивать бета‑версии без разрешения
  | Не пересылать Программу третьим лицам
2.3 - Пользователь подтверждает, что обладает правами на установку ПО на используемом устройстве.
2.4 - Правообладатель не отвечает за действия Пользователя, нарушающие законодательство.
2.5 - Пользователь несёт ответственность за сохранность лицензионного ключа.
2.6 - Правообладатель сохраняет право отозвать лицензию при нарушении условий.
